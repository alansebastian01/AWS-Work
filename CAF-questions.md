## CAF Memorization Shortcut

| Perspective    | Remember                                 |
| -------------- | ---------------------------------------- |
| **Business**   | Strategy, value, projects                |
| **People**     | Skills, training, cloud fluency          |
| **Governance** | Risk, compliance, finance                |
| **Platform**   | Architecture, applications, data         |
| **Security**   | IAM, threat detection, incident response |
| **Operations** | Monitoring, support, service management  |

**Exam shortcut:**
If the question mentions **readiness, transformation, migration planning, organizational change, skills, governance, operations, or security capabilities**, AWS is usually pointing you toward **AWS CAF**.
