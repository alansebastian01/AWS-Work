Since you're already working in data engineering, databases, AWS Glue, ETL pipelines, and cloud-related projects, you can prepare for **AWS Certified Cloud Practitioner (CLF-C02)** much faster than a complete beginner.

## Exam Overview

The CLF-C02 exam has 4 domains: ([AWS Documentation][1])

| Domain                      | Weight |
| --------------------------- | ------ |
| Cloud Concepts              | 24%    |
| Security & Compliance       | 30%    |
| Cloud Technology & Services | 34%    |
| Billing, Pricing & Support  | 12%    |

**Exam Format**

* 65 questions
* 90 minutes
* Passing score: 700/1000
* Multiple choice and multiple response questions ([AWS Documentation][1])

# 14-Day Study Plan

## Days 1-2: Cloud Fundamentals

Learn:

* What is Cloud Computing?
* Benefits of Cloud
* High Availability
* Scalability
* Elasticity
* Fault Tolerance
* Disaster Recovery

Understand:

| Traditional Data Center | AWS Cloud      |
| ----------------------- | -------------- |
| Buy Servers             | Rent Resources |
| CapEx                   | OpEx           |
| Manual Scaling          | Auto Scaling   |

---

## Days 3-4: AWS Global Infrastructure

Must Know:

* Regions
* Availability Zones (AZs)
* Edge Locations
* CloudFront

Example:

* US East (N. Virginia)
* Multiple AZs inside one Region

Questions frequently test:

> Which service improves global content delivery?

Answer:

* CloudFront

---

## Days 5-6: Core Compute Services

### EC2

Know:

* Virtual Servers
* Instance Types
* On-Demand
* Reserved Instances
* Spot Instances

### Lambda

Know:

* Serverless
* Pay per execution

### Elastic Beanstalk

Know:

* PaaS service
* Deploy applications without managing infrastructure

---

## Days 7-8: Storage Services

### S3 (Very Important)

Know:

* Buckets
* Objects
* Storage Classes

Storage Classes:

* Standard
* Intelligent-Tiering
* Standard-IA
* Glacier Instant Retrieval
* Glacier Flexible Retrieval
* Glacier Deep Archive

### EBS

Block storage for EC2

### EFS

Shared file system

Questions on S3 appear heavily in the exam.

---

## Day 9: Databases

Know differences:

| Service     | Type                   |
| ----------- | ---------------------- |
| RDS         | Relational             |
| Aurora      | AWS Managed Relational |
| DynamoDB    | NoSQL                  |
| Redshift    | Data Warehouse         |
| ElastiCache | In-Memory Cache        |

Since you're a data engineer, this section should be easy.

## Day 11: Security (Most Important)

### IAM

Know:

* Users
* Groups
* Roles
* Policies

* Hardware

* Physical Security

Customer Responsible:

* Data
* IAM
* Configurations

### KMS

Encryption

DDoS Protection

### WAF
